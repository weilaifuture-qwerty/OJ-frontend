# Hello World Problem Solutions

## C Solution
```c
#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```

## C++ Solution
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World" << endl;
    return 0;
}
```

## Java Solution
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

## Python 2 Solution
```python
print "Hello World"
```

## Python 3 Solution
```python
print("Hello World")
```

## JavaScript Solution
```javascript
console.log("Hello World");
```
